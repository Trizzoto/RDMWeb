Getting Started

1. Unboxing Your RDM-7 Dashboard

Your package should include:

  • RDM-7 Dashboard
  • 2.5m Power and Ground cable
  • 6 pin Deutsch connector and terminating pins
  • USB-C connection cable (For redundancy troubleshooting)
  • 7.5amp inline fuse if using un-fused power supply
  • Dashboard unit/mounting dimensions

Installation

Wiring Installation

Important: Always ensure battery is DISCONNECTED when making any wiring connections.

Follow these steps to properly wire your RDM-7 Dashboard:

  • Power Connection (Red Wire)
    • Connect to ignition-switched 12V power source
    • Use provided 7.5amp inline fuse if power source is not already fused
  • Ground Connection (Black Wire)
  • CAN Bus Connections
    • CAN High (Brown Wire) - Connect to vehicle's CAN High
    • CAN Low (Grey Wire) - Connect to vehicle's CAN Low
    • Maintain twisted pair configuration for noise reduction

CAN Bus Termination: The RDM-7 Dashboard includes an integrated 120Î© termination resistor. For optimal network performance, we recommend positioning the display at the terminating end of your CAN bus network. If your installation requires removal of the termination resistor, this modification can be performed on the display's circuit board. Please contact our technical support team for detailed guidance on this procedure.

How to Operate

Initial Setup and Operation

After installation, complete these setup steps:

  1. Power up the system with ignition
  2. Verify all connections are secure
  3. Check for proper CAN bus communication
  4. Configure display settings
  5. Test all dashboard functions

Troubleshooting

Common Installation Issues

Display Not Powering On

  • Check ignition power connection
  • Verify ground connection
  • Test inline fuse continuity
  • Measure voltage at power input

No CAN Communication

  • Verify CAN High and Low connections
  • Check for reversed CAN wires
  • Ensure proper CAN bus termination
  • Use USB-C connection for diagnostics

Frequently Asked Questions

What vehicles are compatible with the RDM-7 Dashboard?

The RDM-7 Dashboard is compatible with most modern vehicles equipped with CAN bus systems. Check our compatibility checker for specific vehicle models.

Can I install the dashboard myself?

Yes, the RDM-7 Dashboard is designed for DIY installation. Follow our comprehensive installation guide and wiring diagrams for proper setup.

What if my vehicle doesn't have CAN bus?

Please contact our support team for alternative solutions and adapters that might be available for your specific vehicle.
